# HackerFab IITB

We are a student team at the Indian Institute of Technology Bombay building low-cost open-source student-accessible tools for making transistors.

**Read more at https://hackerfabiitb.github.io/**

This github profile has the following repos our of open-source hardware and software:
[litho-ui](https://github.com/hackerfabiitb/litho-ui): integrated XYZ stage control, camera view and projector control
[litho_machining_m27x1.5](https://github.com/hackerfabiitb/litho_machining_m27x1.5): CNC files for the mechanical components holding the optics
[litho_xyz_stage](https://github.com/hackerfabiitb/litho_xyz_stage): Arduino code to control the NEMA motors of the litho XYZ stage
[mask-designer](https://github.com/hackerfabiitb/mask-designer): design patterns for the litho
